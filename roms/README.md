# ROMs

Legally obtain a Midway Pacman ROM set and place them in this directory,
using the following naming convention:

Program data:
* `pacman.6e` program 0x0000-0x0fff
* `pacman.6f` program 0x1000-0x1fff
* `pacman.6h` program 0x2000-0x2fff
* `pacman.6j` program 0x3000-0x3fff

Video data:
* `82s126.4a` palettes
* `82s123.7f` colors
* `pacman.5e` tiles
* `pacman.5f` sprites

Audio data:
* `82s126.1m` waveforms
* `82s126.3m` audio sequencing (unused)
